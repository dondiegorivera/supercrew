# {{PROJECT_NAME}} — Backlog

Deferred items that are tracked but not yet scheduled into a sprint.
Each item has a stable backlog ID (b-NNN). IDs are never renumbered.

**Statuses:**
- `pending` — ready to schedule when bandwidth allows
- `blocked (reason)` — cannot proceed until a dependency is resolved
- `watching` — monitoring, may not need action
- `deferred (timeline)` — intentionally postponed
